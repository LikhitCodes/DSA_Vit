## Repository Structure

This repository contains all lab assignments for the Data Structures and Algorithms I course. Each assignment is organized in its own folder for easy navigation and reference.

### Folder Organization

Each assignment folder follows this structure:
```
Assignment_XX/
├── assignment_XX.cpp    # C++ source code
└── XX_readme.md         # Documentation with theory, code, and output images
```

### What's Inside Each Folder?

- **C++ Source File (`assignmentXX.cpp`)**: The complete implementation of the assignment
- **README File (`XX_readme.md`)**: Comprehensive documentation including:
  - Theoretical concepts and algorithms
  - Time and space complexity analysis
  - Complete code listing
  - Output screenshots/images
